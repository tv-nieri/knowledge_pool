Sobre esta app

O intuíto desta aplicação, é fornecer um lugar centralizado para compartilhamento de conhecimento de times de Ti. O objetivo principal, é o de mapear onde o time tem mais dificuldades para que uma ação mais precisa seja tomada.

Tecnologias utilizadas:

Python Linguagem de programação

Utilizada por sua boa performance e facilidade de codificação.

Django Web Framework

Utilizado no backend da aplicação por sua simplicidade.

Linux Ubuntu 16.04 Sistema Operacional

Utilizado no deploy por ter python nativo. Também por sua robustez, alta disponibilidade, segurança e confiabilidade.

Gunicorn Middleware (Servidor de aplicação)

Servidor de aplicação do Django. Utilizado por sua boa disponibilidade e integração com Heroku.

Bootstrap Frontend framework

Utilizado por seus estilos serem muito bons, e sua integração com o Django.

Heroku Paas - Platform As A Service

Plataforma de aplicação em nuvem. Utilizado por seu suporte a aplicações Djnago, integração com Git e facilidade de uso.

PostgreSQL Banco de Dados

Utilizado por sua escalabilidade, e ser um dos bancos de dados mais utilizados do momento. Sua licença é gratuíta, o que influencia muito em sua empregabilidade neste projeto.

Pycharm IDE - Integrated Development Environment

Utilizada por ser a melhor IDE do mercado em relação a Python. A ferramenta ainda é integrada com o framework Django (utilizado neste projeto), com o sistema de versionamento Git, e também com o sistema de ambiente virtual Virtualenv.

Git Sistema de versionamento

Utilizado para versional o aplicativo e controlar estas versões. Git é hoje o mais utilizado sistema de versionamento, sendo assim há muito material para consulta na internet, grande motivo de sua adoção neste projeto.
